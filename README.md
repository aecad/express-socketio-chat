# express-socketio-chat

Simple Chat built with Node.js, Socket.io, Express.

## Setup

``` bash
# clone repository
git clone https://github.com/genki-sano/express-socketio-chat.git

# change directory
cd todo-list

# install dependencies
npm i
```

## Usage

```bash
npm start
```

And then point your browser to `localhost:3000`
